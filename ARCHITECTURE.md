# Architecture

## Goal

Build a premium one-page website for a photo studio that feels modern, cinematic, and slightly futuristic.

## Recommended folders

- `app/` — Next.js app router entrypoints
- `components/ui/` — small reusable UI blocks
- `components/3d/` — 3D scene components
- `sections/` — page sections
- `lib/` — content, helpers, constants
- `public/images/` — images and visual assets
- `styles/` — optional extra styles

## Core sections

### Hero
- large headline
- short value proposition
- CTA buttons
- subtle 3D visual on the right or in the background

### About
- what makes the studio special
- equipment, mood, style

### Services
- portraits
- brand shoots
- content days
- creative/editorial sessions

### Portfolio
- image grid or masonry preview
- hover motion

### Pricing
- simple 3-tier cards

### Testimonials
- short social proof quotes

### Booking CTA
- contact form or Telegram/WhatsApp CTA

## 3D ideas

Keep 3D tasteful and light.

Good options:
- floating glass orb
- rotating chrome ring
- soft-lit abstract camera lens object
- layered parallax cards

Avoid heavy scenes early. First version should prioritize style over complexity.

## Motion language

- fade-up on scroll
- soft scale transitions
- low-speed floating motion
- no aggressive particle spam
