# VibeCode, Photo Studio Landing

Pet project for a modern photo studio website with subtle 3D, dark luxury aesthetics, and a premium booking flow.

## Suggested stack

- Next.js (App Router)
- Tailwind CSS
- Framer Motion
- @react-three/fiber
- @react-three/drei

## Project idea

A cinematic landing page for a photo studio with:
- bold hero section
- subtle 3D object or floating glass scene
- services and pricing
- portfolio preview
- booking CTA
- premium dark UI with soft gradients and motion

## Page structure

- Hero
- About studio
- Services
- Portfolio
- Pricing
- Testimonials
- Booking CTA
- Footer

## Design direction

- dark background
- graphite, silver, white, muted violet accents
- large typography
- soft glow, glassmorphism, depth
- smooth transitions, not noisy animation
